This book is written by YouTuber Carlos Azaustre. Carlos is a GDE (Google Developer Expert) and a professional developer. On his channel he has several web programming tutorials and guides, and with this book he walks through all the essentials of JavaScript, reaching up to ES6. While it doesn't go as deep as "You Don't Know JS", it's quite interesting if you're just getting started with this language.
