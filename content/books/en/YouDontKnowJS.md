This book is written by one of the current gurus of the JavaScript world. Kyle Simpson is a true authority. With this series of books you can start with JavaScript from zero up to ES6 while delving into many details that are truly important — and that help you better understand what this language does behind the scenes. It's an almost mandatory read if you've just started with this language. Also, because it's in English, you can learn many terms that may be useful in professional life.
